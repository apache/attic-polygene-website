Envisage

Envisage is a Swing based visualization tool for the Polygene™ Application model. Visualizations can be printed to PDFs.

Usage

Envisage can be easily used directly and prior your Application activation:

public static void main( String[] args )
{
    Energy4Java energy4Java = new Energy4Java();
    ApplicationAssembler assembler = new SchoolAssembler();
    ApplicationDescriptor descriptor = energy4Java.newApplicationModel( assembler );
    new Envisage().run( descriptor );
}

As you can see, Envisage operates on the ApplicationDescriptor, this means that you can easily embed it in your own Applications too.

From the sources you can run the runEnvisageSample or runEnvisageSchoolSample Gradle tasks to run interactive examples. See Build System if you need some guidance.
